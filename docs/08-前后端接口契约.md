# 08 - 前后端接口契约

> v1.0 | 2026-03-30 | 与 brain.json v2.4 同步
>
> 本文档定义企业AI工作站 Phase 1 所有前后端交互的接口规范，包括统一响应格式、分页规范、SSE消息类型、核心API端点签名。

## 一、统一响应格式

### 1.1 成功响应

```typescript
interface ApiSuccessResponse<T> {
  success: true;
  data: T;
  meta?: PaginationMeta;
  requestId: string; // UUID v4，每个请求唯一
}
```

### 1.2 错误响应

```typescript
interface ApiErrorResponse {
  success: false;
  error: {
    code: string;     // 来自 brain.json errorCodes 注册表
    message: string;  // 人可读错误描述（中文）
    details?: any;    // Zod校验失败时为字段级错误数组
  };
  requestId: string;
}
```

### 1.3 示例

**成功 - 单条数据：**
```json
{
  "success": true,
  "data": {
    "id": "conv_01HXYZ",
    "title": "查询本月销售额",
    "createdAt": "2026-03-30T08:15:00.000Z"
  },
  "requestId": "req_a1b2c3d4-e5f6-7890-abcd-ef1234567890"
}
```

**成功 - 分页数据：**
```json
{
  "success": true,
  "data": [
    { "id": "ord_001", "totalAmount": 299.00, "status": "SHIPPED" },
    { "id": "ord_002", "totalAmount": 158.50, "status": "COMPLETED" }
  ],
  "meta": {
    "page": 1,
    "pageSize": 20,
    "total": 95432,
    "totalPages": 4772
  },
  "requestId": "req_f7e6d5c4-b3a2-1098-7654-321fedcba098"
}
```

**错误：**
```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "请求参数校验失败",
    "details": [
      { "field": "page", "message": "必须为正整数" },
      { "field": "startDate", "message": "日期格式无效，需要ISO8601" }
    ]
  },
  "requestId": "req_11223344-5566-7788-99aa-bbccddeeff00"
}
```

## 二、分页规范

### 2.1 分页模式

使用 **offset/limit** 模式（Phase 1 简单优先，不用 cursor）。

### 2.2 请求参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | number | 1 | 页码，从1开始 |
| `pageSize` | number | 20 | 每页条数，最大200 |
| `sort` | string | `createdAt:desc` | 排序，格式 `field:asc\|desc` |

**示例：**
```
GET /api/v1/ai/conversations?page=1&pageSize=20&sort=createdAt:desc
```

### 2.3 分页响应 Meta

```typescript
interface PaginationMeta {
  page: number;       // 当前页码
  pageSize: number;   // 每页条数
  total: number;      // 总记录数
  totalPages: number; // 总页数
}
```

### 2.4 Zod Schema

```typescript
import { z } from 'zod';

const PaginationQuerySchema = z.object({
  page: z.coerce.number().int().min(1).default(1),
  pageSize: z.coerce.number().int().min(1).max(200).default(20),
  sort: z.string().regex(/^\w+:(asc|desc)$/).default('createdAt:desc'),
});
```

## 三、12种AI消息类型（SSE Event Data）

AI对话端点 `POST /api/v1/ai/chat` 使用 SSE（Server-Sent Events）流式输出。每个 SSE event 的 `data` 字段为 JSON，包含 `type` 字段标识消息类型。

### 3.1 text_chunk -- 流式文字片段

```typescript
interface SSETextChunk {
  type: 'text_chunk';
  content: string;       // 文字片段
  messageId: string;     // 消息ID
  index: number;         // 片段序号（从0开始）
}
```

**示例：**
```json
{"type":"text_chunk","content":"本月销售额为","messageId":"msg_001","index":0}
{"type":"text_chunk","content":"¥328,450.00","messageId":"msg_001","index":1}
{"type":"text_chunk","content":"，较上月增长12.3%。","messageId":"msg_001","index":2}
```

**前端渲染：** 实时拼接 content 到聊天气泡中，使用打字机效果逐字显示。

### 3.2 text_complete -- 文字输出完成

```typescript
interface SSETextComplete {
  type: 'text_complete';
  content: string;       // 完整文字内容
  messageId: string;
  tokenUsage: {
    prompt: number;
    completion: number;
    total: number;
  };
}
```

**示例：**
```json
{
  "type": "text_complete",
  "content": "本月销售额为¥328,450.00，较上月增长12.3%。共完成订单4,521笔，客单价¥72.65。时皙品牌贡献了68%的销售额。",
  "messageId": "msg_001",
  "tokenUsage": { "prompt": 850, "completion": 120, "total": 970 }
}
```

**前端渲染：** 替换 text_chunk 拼接的内容为完整文本，停止打字机效果，记录 token 消耗用于成本展示。

### 3.3 tool_call_start -- 工具调用开始

```typescript
interface SSEToolCallStart {
  type: 'tool_call_start';
  toolName: string;      // 工具名称（如 getOrders、getSalesStats）
  toolCallId: string;    // 工具调用唯一ID
  parameters: Record<string, any>; // 调用参数
  messageId: string;
}
```

**示例：**
```json
{
  "type": "tool_call_start",
  "toolName": "getSalesStats",
  "toolCallId": "tc_001",
  "parameters": { "startDate": "2026-03-01", "endDate": "2026-03-31" },
  "messageId": "msg_001"
}
```

**前端渲染：** 显示灰色状态条"正在查询销售数据..."，带加载动画。

### 3.4 tool_call_result -- 工具调用结果

```typescript
interface SSEToolCallResult {
  type: 'tool_call_result';
  toolName: string;
  toolCallId: string;
  result: any;           // 工具返回的结构化数据
  duration: number;      // 执行耗时（ms）
  cached: boolean;       // 是否来自缓存
  messageId: string;
}
```

**示例：**
```json
{
  "type": "tool_call_result",
  "toolName": "getSalesStats",
  "toolCallId": "tc_001",
  "result": {
    "totalAmount": 328450.00,
    "orderCount": 4521,
    "avgOrderAmount": 72.65,
    "topCategory": "护肤品"
  },
  "duration": 1230,
  "cached": false,
  "messageId": "msg_001"
}
```

**前端渲染：** 状态条变为绿色"查询完成(1.2s)"，结果数据可折叠展示。如果 cached=true 显示"(缓存)"标记。

### 3.5 confirmation_card -- 确认卡片（高风险操作）

```typescript
interface SSEConfirmationCard {
  type: 'confirmation_card';
  cardId: string;
  title: string;             // 操作标题
  description: string;       // 操作描述
  riskLevel: 'LOW' | 'MEDIUM' | 'HIGH' | 'CRITICAL';
  changes: Array<{
    field: string;           // 变更字段
    oldValue: string;        // 原值
    newValue: string;        // 新值
  }>;
  confirmText: string;       // 确认按钮文字
  cancelText: string;        // 取消按钮文字
  expiresAt: string;         // 过期时间（ISO8601）
  messageId: string;
}
```

**示例：**
```json
{
  "type": "confirmation_card",
  "cardId": "card_001",
  "title": "商品下架确认",
  "description": "即将下架以下商品，请确认操作",
  "riskLevel": "HIGH",
  "changes": [
    { "field": "商品名称", "oldValue": "-", "newValue": "时皙玻尿酸精华液30ml" },
    { "field": "当前状态", "oldValue": "上架中", "newValue": "已下架" },
    { "field": "影响订单", "oldValue": "-", "newValue": "3笔待发货订单" }
  ],
  "confirmText": "确认下架",
  "cancelText": "取消",
  "expiresAt": "2026-03-30T09:00:00.000Z",
  "messageId": "msg_002"
}
```

**前端渲染：** 渲染为结构化卡片。HIGH/CRITICAL 使用红色边框，LOW/MEDIUM 使用橙色边框。过期倒计时显示。确认/取消按钮调用对应API。

### 3.6 process_indicator -- 过程指示器（多步骤）

```typescript
interface SSEProcessIndicator {
  type: 'process_indicator';
  processId: string;
  steps: Array<{
    name: string;            // 步骤名称
    status: 'pending' | 'running' | 'done' | 'failed';
    detail?: string;         // 步骤详情
  }>;
  currentStep: number;       // 当前步骤索引（从0开始）
  messageId: string;
}
```

**示例：**
```json
{
  "type": "process_indicator",
  "processId": "proc_001",
  "steps": [
    { "name": "解析退货申请", "status": "done" },
    { "name": "查询订单信息", "status": "done", "detail": "订单号 ORD-20260328-0042" },
    { "name": "检查退货条件", "status": "running" },
    { "name": "生成判断结果", "status": "pending" }
  ],
  "currentStep": 2,
  "messageId": "msg_003"
}
```

**前端渲染：** 垂直步骤条，done 显示绿色勾，running 显示蓝色加载动画，pending 显示灰色圆点，failed 显示红色叉。

### 3.7 data_card -- 数据卡片（报表/统计）

```typescript
interface SSEDataCard {
  type: 'data_card';
  cardType: 'kpi' | 'table' | 'chart' | 'list';
  title: string;
  data: any;                 // 卡片类型决定数据结构
  dataSource: string;        // 数据来源标注（防幻觉）

  messageId: string;
}
```

**示例（KPI卡片）：**
```json
{
  "type": "data_card",
  "cardType": "kpi",
  "title": "本月销售概览",
  "data": {
    "metrics": [
      { "label": "销售额", "value": "¥328,450", "change": "+12.3%", "trend": "up" },
      { "label": "订单数", "value": "4,521", "change": "+8.7%", "trend": "up" },
      { "label": "客单价", "value": "¥72.65", "change": "-2.1%", "trend": "down" },
      { "label": "退货率", "value": "3.2%", "change": "+0.5%", "trend": "up" }
    ]
  },
  "dataSource": "ztdy-open OrderPageList API",
  "queryTime": "2026-03-30T08:15:30.000Z",
  "messageId": "msg_004"
}
```

**示例（表格卡片）：**
```json
{
  "type": "data_card",
  "cardType": "table",
  "title": "供应商出货速度排行",
  "data": {
    "columns": ["排名", "供应商", "待发货订单", "平均发货时间"],
```

**前端渲染：** 根据 cardType 渲染对应组件。kpi 渲染为指标卡片网格，table 渲染为表格组件，chart 渲染为图表（ECharts），list 渲染为列表。底部显示 dataSource 和 queryTime。

### 3.8 error -- 错误消息

```typescript
interface SSEError {

}
```

**示例：**
```json
{
  "type": "error",
```

**前端渲染：** 红色错误气泡。recoverable=true 时显示"重试"按钮；recoverable=false 时显示"联系管理员"链接。


### 4.1 认证模块

**请求：**

```typescript
const WechatLoginBodySchema = z.object({
```

### 4.3 AI对话模块

#### POST /api/v1/ai/chat (SSE)

AI对话，流式输出。

**请求：**
```typescript
const AiChatBodySchema = z.object({
  conversationId: z.string().optional(),  // 空则创建新会话
  message: z.string().min(1).max(2000),   // 用户消息
  agentType: z.enum([
    'master', 'operation', 'finance', 'customer_service', 'report', 'system'
  ]).default('master'),                    // 直达模式可跳过 master
});
```

**响应：** SSE 流，event data 为第三节定义的 12 种消息类型

**SSE 典型流程：**
```
data: {"type":"thinking","content":"正在理解您的问题...","messageId":"msg_010"}
data: {"type":"tool_call_start","toolName":"getSalesStats","toolCallId":"tc_010","parameters":{"startDate":"2026-03-01","endDate":"2026-03-31"},"messageId":"msg_010"}
data: {"type":"tool_call_result","toolName":"getSalesStats","toolCallId":"tc_010","result":{"totalAmount":328450},"duration":1230,"cached":false,"messageId":"msg_010"}
```

## 六、通用约定

### 6.1 请求头

| Header | 必需 | 说明 |
|--------|------|------|
| `Authorization` | 认证接口是 | `Bearer <accessToken>` |
| `Content-Type` | POST/PATCH 是 | `application/json` |
| `Accept` | SSE 接口是 | `text/event-stream` |
| `X-Tenant-ID` | 仅登录阶段 | 登录后以 JWT 中的 tenantId 为准 |
| `X-Requested-With` | 是 | CSRF 防护，值为 `XMLHttpRequest` |

### 6.2 日期时间

- 所有时间 **存储为 UTC**
- API 日期参数使用 **ISO 8601** 格式：`2026-03-30T08:15:00.000Z`
- 前端展示按租户时区转换（默认 `Asia/Shanghai`）

### 6.3 排序与过滤

```
# 排序
?sort=createdAt:desc

# 精确过滤
?status=SHIPPED&supplierId=sup_001

# 模糊搜索
?keyword=玻尿酸

# 日期范围
?startDate=2026-03-01T00:00:00.000Z&endDate=2026-03-31T23:59:59.000Z
```

### 6.4 API版本

- 当前版本：`v1`
- 基础路径：`/api/v1`
- 健康检查（公开）：`/health`
